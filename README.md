<br/>

## 更新作業
- `$ npm start`: Vueのサーバー起動
- `$ ./new.sh`: ブログ・実績記事の新規追加
- `$ ./draft.sh`: 下書き中の記事を選択して再開
- `$ ./deploy.sh`: HUGOのビルド、本番公開まで

<br/>

## Files Structure

#### design

#### hugo-amp-contents
- (hugo-amp-contents/README.md)[./hugo-amp-contents/README.md]

#### src
- Vue用のソースファイルです。
